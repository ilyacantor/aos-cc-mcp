# aos-cc-mcp

MCP server exposing Claude Code session logs and scoped write capabilities to the AOS Coordinator.

## Setup

```bash
uv sync
```

## Run tests

```bash
uv run pytest -v
```

## Run server

```bash
uv run aos-cc-mcp
```
